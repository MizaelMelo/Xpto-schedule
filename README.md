# Sistema de agenda #

### Sobre ###

__S__ istema de agenda telefônica.

## Instalação ##

Abra o terminal e localise o arquivo `run.sh` na raiz do projeto. Caso seu sistema operacional seja windows, certifique-se que você possui instalado o git bash em sua máquina.

Rode o seguinte comando:
````bash
$ bash run.sh
````
Isto instalará as dependências, tanto da Api, quanto da aplicação, e também se incubirá de levantar os serviços na porta 8000 e na porta 80, respectivamente.

## Considerações ##
1 - Para fim de facilitar o teste da aplicação, este projeto fora implementado sobre o banco de dados Sqlite e pode ser facilmente localizado em: `./api/database/database.sqlite`.
